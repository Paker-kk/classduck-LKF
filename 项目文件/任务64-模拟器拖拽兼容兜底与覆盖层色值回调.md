# 任务64：模拟器拖拽兼容兜底与覆盖层色值回调

## 1. 任务背景
1. 用户反馈：双提示常驻已通过。
2. 用户仍反馈：模拟器拖拽不稳定，覆盖层过淡接近透明。
3. 要求：继续最小改动，改完重启模拟器并用问询工具收反馈。

## 2. 本次改动

### 2.1 模拟器拖拽兼容兜底
- 文件：`classduck_app/lib/features/schedule/ui/schedule_page.dart`
1. 在黄色容器交互层新增 `Listener.onPointerMove`。
2. 当检测到：
   - 触控/手写笔移动，或
   - 鼠标左键按下移动
   即直接更新扩选区域。
3. 原有 `onPanUpdate` 与 `onLongPressMoveUpdate` 继续保留。

### 2.2 覆盖层色值回调
- 文件：`classduck_app/lib/features/schedule/ui/schedule_page.dart`
1. 覆盖课程区域透明度由 `0.10` 回调为 `0.16`。
2. 模糊仍保持 `sigma 3.0`，维持轻磨砂。
3. 覆盖层使用同一品牌黄，仅做半透明化，不换色。

## 3. 当前交互规则
1. 黄色容器激活时，双提示常驻显示（上下 + 左右）。
2. 外部点击关闭容器。
3. 容器整块点击跳转手动添加。
4. 拖拽扩选支持：普通拖拽、长按拖拽、原生指针拖拽兜底。

## 4. 模拟器复测步骤
1. 点击空白格，出现黄色容器。
2. 鼠标左键按住容器并移动超过一格：
   - 上下移动扩节次。
   - 左右移动扩星期。
3. 若普通拖拽仍偶发失败：长按约 0.4 秒再拖动。

## 5. 验收重点
1. 拖拽是否可稳定触发扩选。
2. 覆盖已有课程时，是否是同品牌黄的半透明效果。
3. 底层课程名/教师/地点是否保持清晰可读。

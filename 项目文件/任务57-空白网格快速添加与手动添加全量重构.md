# 任务57：空白网格快速添加与手动添加全量重构

## 1. 任务背景
1. 用户提出两大改造：
   - 课表主页面空白网格内直接快速添加课程（支持拖拽扩选范围）。
   - 手动添加页面重构为多节次组批量创建模式。
2. 要求保留原有业务链路，不破坏已有课程查看、编辑、删除逻辑。
3. 要求保持品牌视觉一致：品牌黄主操作、白底黑字、统一交互反馈。

## 2. 本次改动

### 2.1 主课表空白网格快速添加
- 文件：`classduck_app/lib/features/schedule/ui/schedule_page.dart`
1. 在课表网格新增空白单元格点击触发：点击后弹出黄色操作容器，默认贴合该单元格。
2. 新增拖拽扩选：
   - 纵向拖拽调节节次范围；
   - 横向拖拽调节星期覆盖范围；
   - 自动限制在网格边界内。
3. 覆盖已有课程时容器切换为半透明玻璃态（不拦截，不回退）。
4. 操作规则：
   - 松手后仅保留选区，不自动跳转；
   - 点击容器中心白色 `+` 才进入手动添加页；
   - 选区外点击会清空选区。
5. 浮动提示：容器右侧新增上下箭头提示，拖拽后或 3 秒后自动隐藏。

### 2.2 选区预填充到手动添加页
- 文件：`classduck_app/lib/features/schedule/ui/schedule_page.dart`
- 文件：`classduck_app/lib/features/schedule/ui/manual_add_course_page.dart`
1. 新增预填充模型：
   - `ManualCoursePrefill`
   - `ManualCourseSessionPrefill`
2. 从选区自动生成多组节次预填充：
   - 每个覆盖星期生成一组配置；
   - 每组自动填充对应星期与起止节次；
   - 上课周数默认 1-16。

### 2.3 手动添加页面重构为多节次组
- 文件：`classduck_app/lib/features/schedule/ui/manual_add_course_page.dart`
1. 保留顶部输入区（课程名称、上课地点、任课教师）。
2. 节次配置改为多组结构，每组独立包含：
   - 单行星期单选
   - 上课节次范围
   - 上课周数范围
   - 卡片颜色选择
3. 新增“新增节次”按钮：点击后复制上一组配置生成新组，持续跟随在最后一组下方。
4. 底部主按钮改为固定悬浮“新建课程”，不随页面滚动离开可视区。
5. 提交逻辑：按节次组数量批量生成独立课程实体，统一共享顶部基础信息。

### 2.4 卡片颜色交互重排
- 文件：`classduck_app/lib/features/schedule/ui/manual_add_course_page.dart`
1. 每组显示 5 个快速颜色 + 同行“更多”入口。
2. 从“更多”选择新颜色后，自动插入快速颜色首位，后续颜色依次后移，保持首页仅展示 5 个。
3. 每组颜色独立维护，互不干扰。

## 3. 验证要点
1. 点击空白格立即出现黄色容器，已有课程格不触发该入口。
2. 拖拽时选区可同步扩展星期和节次，且不会超出网格。
3. 覆盖已有课程时容器呈玻璃态，不阻断滑动。
4. 仅点击容器内 `+` 会进入手动添加页，并正确预填充多组配置。
5. 手动添加页点击“新建课程”后，会按节次组数量生成多条独立课程。
6. 底部“新建课程”按钮在页面滚动时始终可见。

# 任务48：我的二级弹层搜索对齐与分段时间联动修复

## 1. 任务背景
1. 用户复测指出“我的页面”二级弹层仍存在两类问题：搜索文本不居中、搜索栏下方有异常空隙。
2. 用户同时反馈“时间设置 -> 一键批量对齐/时间联动”存在跨上午/下午/晚上串联的问题。
3. 本轮要求继续迭代，不中断联调流程，并补齐项目记录与说明文档。

## 2. 本次改动

### 2.1 我的页面二级弹层搜索区精修
- 文件：`classduck_app/lib/features/profile/ui/profile_page.dart`
1. 两处搜索输入框统一加 `textAlignVertical: TextAlignVertical.center`。
2. 搜索栏改为 `Row(Icon + TextField)` 明确布局，输入框使用 `isCollapsed + contentPadding: EdgeInsets.zero`，彻底修正图标与占位文字垂直偏移。
3. 搜索栏与内容区间距从 8dp 收敛为 4dp。
4. 分组内容容器由 `ListView` 调整为 `SingleChildScrollView + Column`，减少短列表场景下的空白感。

### 2.2 时间编辑联动范围修正
- 文件：`classduck_app/lib/features/schedule/ui/schedule_page.dart`
1. 侧栏时间编辑的“后续节次顺延”改为仅作用于当前时段。
2. 新增时段终点判定：
   - 上午节次仅顺延到 `morningEnd`
   - 下午节次仅顺延到 `afternoonEnd`
   - 晚上节次仅顺延到 `eveningEnd`
3. 保持现有保存逻辑与数据结构不变，仅修正联动边界。

### 2.3 一键批量对齐锚点固定
- 文件：`classduck_app/lib/features/schedule/ui/schedule_page.dart`
1. 批量对齐改为三段固定锚点：上午 `08:00`、下午 `14:00`、晚上 `19:00`。
2. 每段内部按“课时时长 + 课间时长”顺延，段与段之间不再互相继承结束时间。
3. 解决了“下午首节贴到上午末节后”的跨时段串联问题。

## 3. 验证
1. 目标文件已执行静态错误检查：`profile_page.dart` 与 `schedule_page.dart` 均无分析错误。
2. 计划在安卓模拟器重新部署后，重点回归以下路径：
   - 我的页面 -> 已上课程/已完成待办/今日状态（二级弹层搜索对齐与间距）
   - 课表侧栏 -> 时间设置 -> 一键批量对齐（时段内联动边界）

## 4. 验收关注点
1. 搜索框占位文本和输入文本是否在灰色容器内垂直居中。
2. 搜索栏与首条内容之间是否还存在突兀空白。
3. 点击一键批量对齐后，下午和晚上的首节是否仍保持各自时段起点，不再贴到上午末节。

# 任务63：覆盖层可见度回调与拖拽兜底手势补齐

## 1. 任务背景
1. 用户反馈：双提示常驻已通过。
2. 用户新增反馈：
   - 覆盖已有课程时，覆盖层过淡，观感接近透明。
   - 安卓模拟器中按住拖动不稳定，希望补充可执行方案。
3. 约束：继续最小改动，不动已通过交互。

## 2. 本次改动

### 2.1 覆盖层可见度微调
- 文件：`classduck_app/lib/features/schedule/ui/schedule_page.dart`
1. 覆盖课程单元格透明度从 `0.10` 回调到 `0.16`。
2. 模糊强度维持 `sigma 3.0`，继续保持轻磨砂与高可读性。

### 2.2 拖拽兜底手势
- 文件：`classduck_app/lib/features/schedule/ui/schedule_page.dart`
1. 保留原 `onPanUpdate` 拖拽扩选。
2. 新增 `onLongPressMoveUpdate` 作为兜底：
   - 长按后拖动也可扩选；
   - 便于在模拟器下规避手势识别差异。

## 3. 安卓模拟器建议操作
1. 先点击空白格，出现黄色容器。
2. 优先方式：按住鼠标左键直接拖动（上下/左右）。
3. 兜底方式：按住不动约 0.4 秒后再拖动（长按拖动）。
4. 若仍无变化：确保是拖动黄色容器本体，不是滚轮，也不是拖窗口边框。

## 4. 验收点
1. 覆盖课程区域应有“轻黄可见提示”，不再近似透明。
2. 底层课程名/教师/地点仍可清晰读取。
3. 直接拖动或长按拖动，至少一种方式可稳定触发扩选。

# 任务59：阻断级按钮修复与黄色容器交互全量收口

## 1. 任务背景
1. 手动添加页出现阻断级问题：`新建课程` 按钮错误覆盖输入区，影响 `上课地点` 等字段输入。
2. 课表黄色快速添加容器存在误触循环、热区过小、引导信息不足、覆盖既有课程视觉粗糙等问题。
3. 目标：在不改动核心业务链路前提下，做最小范围修复并完成交互收口。

## 2. 本次改动

### 2.1 紧急 BUG1：手动添加页底部按钮位置/图层/键盘适配
- 文件：`classduck_app/lib/features/schedule/ui/manual_add_course_page.dart`
1. `Scaffold` 增加 `resizeToAvoidBottomInset: false`，避免系统与自定义位移同时生效导致按钮错位。
2. 底部按钮位置改为：
   - 常态：`safeBottom + 12`（紧贴底部安全区上方）
   - 键盘弹起：`viewInsets.bottom + 12`（自动抬升到键盘上方）
3. 列表底部内边距改为 `136 + bottomInset`，确保键盘弹起时输入内容和按钮都可见，不互相覆盖。
4. 保持原设计样式：黄底、白字、满宽圆角，不改视觉规范仅修正布局行为。

### 2.2 黄色容器外部点击关闭逻辑修复（误触循环拦截）
- 文件：`classduck_app/lib/features/schedule/ui/schedule_page.dart`
1. 容器激活时新增全屏关闭层（位于课程卡之上、容器之下）。
2. 点击容器外任意区域仅执行 `_clearQuickAddSelection()`，不会触发新的容器弹出。
3. 同步关闭容器及其滑动引导提示，页面恢复无激活容器状态。

### 2.3 黄色容器整块热区点击跳转
- 文件：`classduck_app/lib/features/schedule/ui/schedule_page.dart`
1. 容器由“仅 + 图标可点”改为“整个黄色容器区域可点”。
2. 容器 `onTap` 统一触发 `_openManualAddFromQuickSelection()`。
3. 保留滑动与点击隔离：
   - `onPanStart/onPanUpdate` 仅做选区调节与提示隐藏
   - 滑动手势不会触发跳转

### 2.4 滑动引导提示增强
- 文件：`classduck_app/lib/features/schedule/ui/schedule_page.dart`
1. 提示保持在容器右侧半透明白色圆角浮层。
2. 提示内容改为同时展示：`上下箭头 + 左右箭头`。
3. 显示/消失规则：
   - 容器出现时显示
   - 开始滑动立即隐藏
   - 3 秒未滑动自动隐藏
   - 容器关闭同步隐藏

### 2.5 覆盖已有课程的磨砂玻璃态优化
- 文件：`classduck_app/lib/features/schedule/ui/schedule_page.dart`
1. 选区内按单元格渲染：
   - 未覆盖课程单元格：100% 不透明品牌黄
   - 覆盖课程单元格：`BackdropFilter` + 品牌黄 `40%` 透明度
2. 视觉上形成覆盖与未覆盖区域清晰对比，同时保留下层课程可读性。

## 3. 关键实现位置
1. 手动添加页按钮键盘适配：
   - `build` 内 `bottomInset/safeBottom/actionBottom` 计算
   - `Positioned(bottom: actionBottom)`
2. 黄色容器外部关闭层：
   - 课表网格 `Stack` 内 `if (showSelection) Positioned.fill(GestureDetector...)`
3. 黄色容器整块点击：
   - 选区 `GestureDetector(onTap: _openManualAddFromQuickSelection)`
4. 单元格级磨砂渲染：
   - 选区 `Stack` 中按 day/period 双循环绘制每格状态

## 4. 滑动功能测试操作说明

### 4.1 安卓模拟器（鼠标）
1. 上下滑动扩选：
   - 先点击空白网格单元格，出现黄色容器。
   - 鼠标左键按住黄色容器不放，向上/向下拖动后松开。
2. 左右滑动扩选：
   - 鼠标左键按住黄色容器不放，向左/向右拖动后松开。
3. 关键区别：
   - 仅“左键按住拖动”是有效滑动手势。
   - 鼠标滚轮只会滚动页面，不会触发容器范围调节。

### 4.2 真机（触控）
1. 单指按住黄色容器并上下拖动：调节节次范围。
2. 单指按住黄色容器并左右拖动：调节星期范围。
3. 拖动结束后，单击黄色容器任意位置：进入手动添加页。

### 4.3 判定标准
1. 容器弹出后，右侧提示显示双向箭头；滑动后或 3 秒后消失。
2. 容器激活时，点击容器外只关闭当前容器，不会新弹一个容器。
3. 容器覆盖已有课程时，覆盖格为 40% 透明磨砂黄；未覆盖格保持纯黄。
4. 手动添加页中：
   - 键盘未弹起：`新建课程` 固定在底部安全区上方
   - 键盘弹起：按钮抬升到键盘上方且不覆盖输入框
   - 键盘收起：按钮回到底部初始位置
